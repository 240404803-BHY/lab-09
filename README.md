TravelApp – Seyahat Rehberi Uygulaması

Bu çalışma, Mobil Programlama dersi kapsamında verilen React Navigation Seyahat Rehberi Uygulaması ödevi için geliştirilmiştir.
Uygulamada React Navigation kullanılarak çoklu ekran yapısı oluşturulmuş, Stack ve Tab Navigator yapıları entegre edilmiş ve ekranlar arasında veri iletimi sağlanmıştır.

Tüm adımlar sırasıyla uygulanmış olup, her bölüm için ayrı commit mesajları Git geçmişine eklenmiştir.

---

Projeyi Çalıştırma Rehberi

Gerekli Kurulumlar:
-Sistemde Node.js ve npm'in yüklü olması gerekmektedir.
-Mobil cihazda Expo Go uygulaması bulunmalıdır.
-İstenirse Android veya iOS emülatörü de kullanılabilir.
